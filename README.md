**Diaww-iCard** adalah sebuah kartu identitas digital (Digital Business Card) berbasis web yang modern, responsif, dan elegan. Cocok digunakan untuk personal branding, portofolio singkat, atau sebagai halaman "Link in Bio" untuk media sosial kamu.

---

## ✨ Fitur Utama

* 📱 **Responsive Design**: Tampilan sempurna di semua perangkat (HP, Tablet, PC).
* 🎨 **Modern UI/UX**: Desain minimalis dengan efek Glassmorphism yang kekinian.
* 🔗 **Social Media Integration**: Hubungkan semua platform media sosial kamu dalam satu tempat.
* ⚡ **Fast Loading**: Ringan dan dioptimalkan untuk performa maksimal.
* 🛠️ **Easy Customization**: Sangat mudah untuk mengubah data diri dan ikon.

---

## 🚀 Teknologi yang Digunakan

* **HTML5** - Struktur dasar.
* **CSS3** - Styling & Animasi (Custom Properties/Variables).
* **JavaScript (Optional)** - Interaktivitas.
* **FontAwesome / BoxIcons** - Ikon sosial media yang keren.

---

## 🛠️ Cara Kloning & Penggunaan

Jika kamu ingin menggunakan proyek ini di komputer lokal atau HP kamu:

1.  **Clone Repository**
    ```bash
    git clone [https://github.com/diaz1414/Diaww-iCard.git](https://github.com/diaz1414/Diaww-iCard.git)
    ```
2.  **Masuk ke Direktori**
    ```bash
    cd Diaww-iCard
    ```
3.  **Buka file `index.html`** di browser favorit kamu.

---

## ⚙️ Kustomisasi

Untuk mengubah informasi di dalam kartu:
1. Buka file `index.html`.
2. Cari bagian teks (seperti Nama, Deskripsi, Link).
3. Ganti sesuai keinginan kamu.
4. Simpan dan *Push* kembali ke GitHub!
